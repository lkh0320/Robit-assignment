# 🌤️ 날씨 정보 조회 프로그램

## 1. 프로젝트 소개

Python과 OpenWeather API를 이용하여 특정 도시의 날씨 정보를 조회하는 프로그램입니다.

사용자가 도시를 입력하면 조회 가능한 날짜를 출력하고, 원하는 날짜를 선택하면 해당 날짜의 날씨 정보를 확인할 수 있습니다.

---

## 2. 개발 목적

- Python에서 외부 API를 사용하는 방법을 학습한다.
- requests 라이브러리를 이용하여 인터넷에서 데이터를 받아오는 방법을 익힌다.
- JSON 데이터를 분석하여 필요한 정보를 추출하는 방법을 배운다.
- 사용자 입력과 반복문, 조건문을 활용하는 방법을 익힌다.

---

## 3. 개발 환경

- Python 3.x
- Visual Studio Code
- OpenWeather API
- requests 라이브러리

---

## 4. 사용 라이브러리

### requests

requests는 Python에서 HTTP 요청(GET, POST 등)을 쉽게 보낼 수 있도록 만들어진 라이브러리이다.

이번 프로젝트에서는 OpenWeather API 서버에 요청을 보내고 날씨 정보를 받아오기 위해 사용하였다.

설치 방법

```bash
pip install requests
```

사용 예시

```python
import requests

url = "https://api.openweathermap.org/data/2.5/forecast"

response = requests.get(url, params=data)

weather = response.json()
```

주요 함수

| 함수 | 설명 |
|------|------|
| requests.get() | 서버에 GET 요청을 보낸다. |
| response.json() | JSON 데이터를 Python 딕셔너리 형태로 변환한다. |

---

## 5. 프로그램 기능

- 도시 입력
- 조회 가능한 날짜 출력
- 원하는 날짜 선택
- 해당 날짜의 날씨 조회
- 기온 출력
- 최고기온 출력
- 최저기온 출력
- 습도 출력
- 날씨 상태 출력

---

## 6. 실행 방법

1. OpenWeather 사이트에서 API Key를 발급받는다.
2. 코드의 API Key 부분에 자신의 키를 입력한다.
3. 프로그램을 실행한다.
4. 도시를 입력한다.
5. 조회 가능한 날짜 중 원하는 날짜를 입력한다.
6. 해당 날짜의 날씨 정보를 확인한다.
